Marvel Metadata: Unmasking Heroes with Data

Master's of Data Science Capstone Project

Overview

This project explores the metadata of Marvel comic characters through large-scale web scraping, text classification, and relational database design. By extracting and analyzing biographies and descriptions from Marvel’s character pages, the goal was to build a structured and searchable database of hero names, powers, affiliations, and thematic elements. The project also surfaces underrepresented characters and storylines through keyword-based tagging and classification.

Skills & Tools Used

  • Python (BeautifulSoup, requests, pandas, regex)
  • PostgreSQL (database schema, joins, indexing)
  • SQL (queries, classification tables, foreign keys)
  • Regex-based NLP techniques for text classification
  • GitHub for project tracking and iterative refinement

Process

  • Scraping: Collected all character URLs from Marvel’s site and scraped each profile’s name, aliases, powers, biography, and affiliations.
  • Parsing: Cleaned and parsed structured and unstructured data using regular expressions and conditional logic.
  • Classification: Tagged comics and characters with attributes like theme, threat level, and event involvement using curated keyword dictionaries.
  • Database Design: Created normalized tables for characters, comics, teams, threats, and themes using PostgreSQL with primary/foreign key integrity.
  • Expansion: Iteratively improved keyword lists and classification coverage, reaching over 90% theme-tag coverage on the dataset.

Key Outcomes

  • Scraped and structured data for over 27,000 Marvel comics and hundreds of character profiles
  • Built SQL tables for teams, powers, aliases, themes, and threats, supporting fast lookup and filtering
  • Used regex-based keyword classification to tag comics with major themes and threat levels
  • Identified gaps in Marvel’s character representation and uneven metadata across character profiles
  • Achieved over 90% coverage of themes across the comic dataset

Reflection

This project was a challenging and rewarding deep dive into messy real-world text data and the power of structured storage. I learned how to combine scraping, regex, and SQL to extract meaning from unstructured HTML. The iterative keyword design taught me a lot about classification tradeoffs and maintaining reproducibility. If extended, I’d love to add visualizations for character connections, theme evolution over time, or integrate sentiment analysis on story arcs.
